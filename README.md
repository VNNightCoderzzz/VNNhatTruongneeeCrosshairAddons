## 🌟 Introduction
**VNNhatTruongneee's Crosshair Addons** is developed by **VNNhatTruongneee**. This mod focuses on improving the user experience through intelligent visual feedback directly on the crosshair, helping you better control distance and targets.

## 🔥 Key Features

### ⚔️ Smart Combat Indicator
This is the main feature that helps you never miss a decisive shot:
* **Automatic color change:** The crosshair will turn **bright red** when you look at another player.

* **Reach Check:** Red only appears when the target is within attack range.

* **Benefit:** Helps you know exactly when to click without having to guess the distance.


## ⚙️ System Requirements
For optimal mod performance on version **1.21.11**, you need to install the following dependencies:
1. [Fabric Loader](https://fabricmc.net/)
2. [VNNhatTruongneeeLib](https://modrinth.com/) (Required)
